# locale

## dir

- Type: `string`
- Default: `src/locales`

语言包的目录设置

## defaultLocale

- Type: `string`
- Default: `en-US`

默认语言设置

## extensions

- Type: `string[]`
- Default: `['js', 'ts']`

默认支持的文件类型.

## exclude

- Type: `string[]`
- Default: `[]`

默认会加载文件中需要排除的文件列表.

使用指南请[参考多语言](/convue/guide/locales)。
